# pytorch-udemy

Notebooks and notes from a PyTorch Udemy course, worked through in Google Colab.

## Workflow (Colab ↔ GitHub)

This repo is the GitHub destination for the course's Colab notebooks. Colab reads
from and writes to it directly — no local cloning needed to follow along.

**Save a notebook to this repo (from Colab):**
`File → Save a copy in GitHub` → pick `luoshuaiqing/pytorch-udemy` → set a path
(e.g. `lectures/section-03-cnns.ipynb`) → commit.

**Reopen any notebook later:**
`File → Open notebook → GitHub tab` → enter `luoshuaiqing/pytorch-udemy` →
pick from the list of every `.ipynb` in the repo.

**Open a specific notebook by URL** (swap the domain):
`github.com/luoshuaiqing/pytorch-udemy/blob/main/<path>.ipynb`
→ `colab.research.google.com/github/luoshuaiqing/pytorch-udemy/blob/main/<path>.ipynb`

## Notes

- Saved notebooks include cell outputs. To keep diffs clean, run
  `Edit → Clear all outputs` before the final save (optional — outputs are also a
  useful record of progress).
- One repo per course keeps things separate; switch repos in Colab's Open dialog.
